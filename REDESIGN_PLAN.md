# Modern Dark Theme Website Redesign Plan

## Objective
Complete website redesign with modern, premium dark theme and professional UI

## Design Specifications
- **Theme**: Deep black/charcoal background with neon accents
- **Colors**: Neon blue, electric purple, cyan, gradient glow effects
- **Typography**: Modern sans-serif with proper hierarchy
- **Effects**: Glassmorphism, smooth animations, hover glow effects
- **Layout**: Clean, futuristic, fully responsive

## New Structure
### Pages to Create/Update:
1. **index.html** - Modern hero landing page
2. **about.html** - About section with skills/education
3. **services.html** - Services offered
4. **portfolio.html** - Projects showcase
5. **blog.html** - Learning resources (existing, will update)
6. **contact.html** - Contact form and information
7. **roadmap.html** - Web dev roadmap (existing, will update)

### Key Features:
- Glassmorphism cards with backdrop blur
- Gradient backgrounds and neon glow effects
- Smooth scroll and micro-interactions
- Modern navigation with hover effects
- Responsive grid layouts
- Animated icons with glow on hover
- Professional typography hierarchy

## Implementation Steps
1. Create new modern CSS framework
2. Update index.html with hero landing page
3. Create new pages (about, services, portfolio, contact)
4. Update existing pages (blog, roadmap)
5. Add JavaScript for animations and interactions
6. Test responsiveness and polish

## Color Palette
- **Primary Background**: #0a0a0a (deep black)
- **Secondary Background**: #1a1a1a (charcoal)
- **Glass Background**: rgba(255, 255, 255, 0.05) with backdrop-blur
- **Neon Blue**: #00f5ff
- **Electric Purple**: #8b5cf6
- **Cyan**: #06b6d4
- **Text Primary**: #ffffff
- **Text Secondary**: #a1a1aa

## Typography
- **Font Family**: 'Inter', 'SF Pro Display', system-ui
- **Headings**: Bold, gradient text effects
- **Body**: Clean, readable line-height
- **Hierarchy**: H1 (4rem) → H2 (2.5rem) → H3 (1.8rem) → Body (1rem)

Module 1 – Getting Started with HTML & CSS

🧱 HTML – The Skeleton of Your Webpage

Imagine building a house.

  • 🧱 HTML → Structure
  • 🎨 CSS → Design
  • ⚡ JavaScript → Functionality

HTML provides the blueprint and structure of the webpage.

📖 What is HTML?

HTML = HyperText Markup Language

✔ Used to structure web pages
✔ Uses tags to define elements

HTML is NOT a programming language. It is a markup language.

🧩 HTML Tags

Tags tell the browser how content should appear.

Try it yourself

🔹 Self Closing Tags

Some tags do not need closing tags.

Tag Purpose
<br> Line break
<hr> Horizontal line
<img> Image
